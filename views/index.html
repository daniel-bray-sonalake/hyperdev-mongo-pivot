Hyperdev, mongo and pivots

A sample of using node to pull data from mongo, transform it into a very simple table format, so a javascript component in the browser can pivot the data as the user desires

This is mostly a demonstration of how to use hyperdev.com to pull data from an external data source (in this case mongodb running on mlab.com) and present it in a simple UI.

Why pivots? We all go a little mad sometimes, and I do love pivots
